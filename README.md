# Trabajo Práctico 1: Variables en JavaScript

Este repositorio contiene el primer trabajo práctico enfocado en el uso de variables en JavaScript, desarrollado para la comisión Web12. El objetivo es reforzar los conceptos fundamentales de las variables y su manipulación en el lenguaje.

## Como ejecutar el proyecto 💻

1. Clona el repositorio: 
`git clone https://github.com/ExequielRojas/tp1-Variables.git`

1. Abrir la carpeta creada y ejecutar el index.html

## 📌 Requisitos

- Un navegador web actualizado (como Chrome, Firefox o Edge).
- Visual Studio Code u otro editor de código de tu preferencia para visualizar y editar los archivos JavaScript.

## 🚀 Tecnologias utilizadas 🚀

Las herramientas que se usaron en el proyecto son las siguientes:
